------------------

1. data/
   - __init__.py          : Package init
   - loader.py            : NSL-KDD data pipeline with synthetic fallback
                           - Downloads NSL-KDD or generates synthetic data
                           - Preprocessing with LabelEncoder + MinMaxScaler
                           - Partition for federated learning across offices

2. core/
   - models.py            : ML models (XGBoost/RandomForest)
                           - FederatedModel class with weight extraction
                           - Train, predict, evaluate methods
                           - Save/load model functionality
   - federated.py         : Federated Learning orchestration
                           - FL loop with local training
                           - Krum Byzantine filtering
                           - Differential Privacy (Laplace noise)
                           - Round metrics tracking
   - evaluation.py        : Experiment evaluation tools
                           - ExperimentEvaluator: tracks experiments
                           - AttackAnalyzer: detects & logs attacks
                           - ReputationTracker: office reputation
   - web3_client.py       : Blockchain integration
                           - BlockchainClient with web3.py
                           - ReputationManager for slashing/rewards
                           - Audit trail generation

3. reports/
   - generator.py         : Report generation
                           - HTML report with tables/charts
                           - PDF report (if reportlab installed)
                           - Loads data from logs/

4. main.py                : Main entry point
                           - CLI with modes: baseline, byzantine, dp, reputation, full
                           - --n-offices, --rounds, --attack-office, --epsilon args

5. requirements.txt       : Dependencies

6. dashboard.py           : Streamlit dashboard (updated)
                           - 4 tabs: FL Metrics, Byzantine Defense, Reputation, Audit
                           - Plotly charts for visualization
